---------------------

  * Summary
  * Installation

  * Credits


SUMMARY
-------

This module allows admins to configure user account creation verification by 
verification code.

This module simply changes the user verification flow.


INSTALLATION
------------

1. Follow steps at https://www.drupal.org/node/1897420


CONFIGURATION
-------------
Configure the module at the configuration page,  Admins can set number of
digits in the verification code, number of times user can request OTP, number
of times user can use the verify otp form in a given time frame, etc.

CREDITS
-------

The idea came up from gokhalemethod.com requiring OTP verification and no
module providing it for free.
